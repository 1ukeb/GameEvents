# TinyTools
* Lightweight collection of easy to use Unity tools
* Scriptable Events
* Dynamic Audio System
* Unity Type Extensions

### Getting Started
* Each tool contains a demo folder with a basic demo and instructions.
* Unity Version 2018.1.0f1 or higher

### Version History
